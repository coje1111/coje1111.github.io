# 2026-01-27

## 오늘 한 것
- Java 사고훈련(Level 1)을 말로만 진행해봄
- 문제를 코드 없이 사고로 풀어보는 방식을 시도
- 기존 사고훈련 방식이 나에게 맞는지 점검
- 앞으로의 학습 방향을 다시 고민함

## 느낀 점 / 막힌 점
- 문제를 말로 풀다 보니  
  사고가 깊어지기보다는 문제를 그대로 반복하는 느낌이 들었음
- “생각하는 훈련”이라기보다는  
  이미 알고 있는 개념을 말로 바꾸는 느낌이 강했음
- 이 방식이 지금 내 사고력을 키우는 데  
  직접적인 도움이 되는지 의문이 들었음

## 정리
- 현재 단계에서는 사고훈련보다  
  **아주 쉬운 코드 풀이를 통해 흐름을 익히는 게 더 필요하다고 판단**
- 사고훈련을 완전히 버리기보다는  
  지금은 잠시 멈추고 다른 방식으로 접근하기로 결정
- 방향을 점검한 날로 의미를 두고 기록만 남김

## 다음 계획
- 쉬운 코드 풀이 문제부터 다시 시작
- “생각 → 코드 → 결과” 흐름을 눈으로 확인하는 연습에 집중
