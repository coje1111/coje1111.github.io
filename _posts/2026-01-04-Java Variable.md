# 2026-01-04

## 오늘 한 것
- 조건문(if / else if / else)을 이용한 할인 계산 문제 집중 연습
- 할인율을 먼저 결정하고, 이후 계산하는 패턴 반복 연습
- 문자열 비교 시 equals() 사용 연습

## 막힌 점
- 변수 선언 위치와 중괄호 범위(scope)에서 여러 번 컴파일 에러 발생
- 조건문 안에서 변수를 다시 선언해 에러가 나는 경우가 있었음
- 뼈대가 주어진 문제는 푼 것처럼 느껴졌지만,
  실제로는 코드를 처음부터 전부 작성하지 못했다는 점을 인식함

## 해결
- 변수를 main 메서드 상단에서 한 번만 선언하고,
  조건문에서는 값만 변경하는 방식으로 수정
- 모든 실행 코드는 class와 main 메서드 내부에 있어야 한다는
  자바 기본 구조를 다시 확인
- 조건문 문제를 워밍업용과 실전용으로 구분해서 학습하기로 결정
