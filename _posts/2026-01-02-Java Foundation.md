## 오늘 한 것
- Java 변수 선언과 계산 문제 연습
- 할인율 계산 과정에서 int / double 형변환 연습
- Git 기본 흐름(init, add, commit, push) 직접 실습

## 막힌 점
- 홈 디렉토리에서 git init을 실행해서
  시스템 파일들이 전부 Git 추적 대상이 되는 문제 발생
- 원격 저장소(origin)를 연결하지 않은 상태에서 push 시도
- 폴더 구조가 복잡해져서 어떤 폴더가 레포인지 헷갈림

## 해결
- 홈 디렉토리의 .git 삭제 후 프로젝트 폴더에서 다시 init
- GitHub에서 레포 생성 후 remote add origin으로 연결
- dev-root 기준으로 개인 / 강의 / 팀 프로젝트 폴더 구조 정리
